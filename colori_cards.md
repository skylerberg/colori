# Colori Cards Reference

## Card Actions Glossary

| Action | Description |
|--------|-------------|
| **Make Project** | Triggers the project-making process: spend colors from your wheel and a material to complete a project card |
| **Make Materials ×N** | Convert N cards from your hand into material tokens |
| **Mix Colors ×N** | Combine N pairs of colors on your wheel into new colors |
| **Draw Cards ×N** | Draw N cards from your deck |
| **Destroy Cards ×N** | Permanently destroy N cards from your hand (removes them from the game) |

# Color Mixing Reference

Colors are arranged on a 12-color wheel. Valid mixes are:
- Any two different **primaries** produce a **secondary**
- A **primary** and an adjacent **secondary** (2 steps apart on the wheel) produce a **tertiary**

| Input A | Input B | Result |
|---------|---------|--------|
| Red | Yellow | Orange |
| Yellow | Blue | Green |
| Red | Blue | Purple |
| Red | Orange | Vermilion |
| Yellow | Orange | Amber |
| Yellow | Green | Chartreuse |
| Blue | Green | Teal |
| Blue | Purple | Indigo |
| Red | Purple | Magenta |

# Game Setup

1. **Player decks** — Each player receives a 10-card starting deck (2 Basic Red, 2 Basic Yellow, 2 Basic Blue, 1 Glass, 1 Textiles, 1 Ceramics, 1 Paintings), shuffled.
2. **Draft deck** — Shuffle all 76 draft cards (60 dye + 16 material) into a single face-down deck.
3. **Project deck** — Shuffle all 60 project cards into a face-down deck.
4. **Project display** — Deal 6 project cards face-up from the project deck.
5. **Round** — Game starts at round 1 in the draw phase.
